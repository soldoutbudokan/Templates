# Figures

Reusable economics and data visualization figures with reproducible code.

## Contents

### Deadweight Loss (DWL) Chart
A standard supply-and-demand diagram illustrating deadweight loss from a price increase (P₁ → P₂), showing consumer DWL, producer DWL, and surplus transfer regions.

![DWL Chart](DWL%20Chart/output/DWL_chart.png)

**Quick Start:**
```bash
cd "DWL Chart"
python code/DWLchartcode.py
```
Output saves to `DWL Chart/output/DWL_chart.png`.
